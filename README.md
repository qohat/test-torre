# test-torre
Spring boot kotlin - Reactor Project - JS Client Angular 8.

I decided create an Reactive aplication using SpringBoot Koltin, Reactor Project in Backend. My DB will be H2 in memory.
In front end i am using angular 8 with Material Design.

I am designing the app with reactive extensions in front and backend.

In backend I use the DDD pattern and CQRS.

And with reactor, i am using NIO server like Netty for the reactive streams.

--------------
The app consist in a cliente to promote inexperienced professionals, taking advantage of the skills they want to improve.

Basically the app get the profile name of user and get the interests from his bio, with a Reactive Web Client.

--------------

With the interests loaded, a reactive form dinamic is generated, please fill the form and continue.

---- To restart with another user, use de side nav and go to home.
